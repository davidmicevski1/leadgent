# Initial Task Board (First 30 Days)

Use this as your active execution board.

## Phase A: Offer and Positioning

- [ ] Choose one ICP and one niche pain to own
- [ ] Finalize one flagship service and one add-on service
- [ ] Define offer scope, exclusions, timeline, and pricing floor
- [ ] Create a short qualification checklist for inbound/outbound prospects

## Phase B: Lead and Campaign System

- [ ] Define lead sourcing channels and list criteria
- [ ] Define mandatory enrichment fields and QA checks
- [ ] Draft 2 outbound sequence variants
- [ ] Prepare personalization framework for first touch
- [ ] Launch first LeadGent acquisition campaign

## Phase C: Delivery System

- [ ] Finalize client onboarding template
- [ ] Finalize weekly reporting template
- [ ] Create delivery SOP for weekly execution cycle
- [ ] Create escalation checklist for blocked campaigns

## Phase D: Proof and Growth

- [ ] Close first client with clear scope
- [ ] Deliver first measurable result and document before/after
- [ ] Turn result into one case study asset
- [ ] Capture lessons learned and update templates/SOPs

## Weekly Check-In Questions

- [ ] What created pipeline progress this week?
- [ ] What broke in delivery and how did we fix it?
- [ ] What is the single highest-leverage action next week?
