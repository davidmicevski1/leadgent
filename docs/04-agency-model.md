# Agency Model and Service Design

## Service Packages

1. Outbound Setup Sprint (2-3 weeks)
- ICP targeting
- Data sourcing and enrichment workflow
- Messaging base sequence
- Infrastructure and tracking setup

2. Outbound Execution Retainer (monthly)
- List building and QA
- Sequence execution and optimization
- Weekly reporting and strategy calls

3. GTM Automation Sprint (project or add-on)
- Workflow mapping
- Tool integration and automation build
- Monitoring + handover documentation

## Delivery Capacity Rule

- Start with max 3 active clients at once.
- Grow only when delivery QA is stable and documented.

## Pricing Logic (Framework)

- Setup fee for foundational implementation
- Monthly retainer for execution and optimization
- Optional performance bonus when target outcomes are hit

## Client Success Guardrails

- Weekly KPI report is mandatory
- Clear owner for each client stream
- Red-flag account when no positive movement over 2 weeks
