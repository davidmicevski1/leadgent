# LeadGent Operating System

## Weekly Rhythm

Monday:
- Set top 3 priorities for pipeline, delivery, and operations
- Confirm client-facing and internal execution blocks

Tuesday-Thursday:
- Execute sourcing, campaigns, copy iterations, and delivery work
- Document bottlenecks and process fixes

Friday:
- Send client updates
- Run internal retrospective
- Lock next week's priorities

## Decision Framework

1. Client outcome impact
2. Revenue impact
3. Speed to execute
4. Reusability across accounts

## Execution Triggers (No hard metrics yet)

- Weak campaign signal for two consecutive weeks: change list + angle
- Poor lead quality: tighten sourcing criteria and enrichment QA
- Low meeting quality: improve qualification and offer clarity
- Delivery friction across accounts: standardize the workflow into template/SOP

## Documentation Rule

If a task repeats twice, convert it into a template, checklist, or automation.
