# Starting Backlog (Agency Launch)

## Week 1

1. Finalize ICP and primary service positioning.
2. Write one-page offer and pricing sheet.
3. Build source list criteria and enrichment field set.
4. Draft first outbound sequence for LeadGent acquisition.
5. Define QA checklist for lead and personalization quality.

## Week 2

1. Launch campaign to first target segment.
2. Build onboarding packet and kickoff checklist.
3. Create weekly client report format and update cadence.
4. Build first case-study template.
5. Create API reporting requirements doc (for later implementation).

## Week 3-4

1. Run A/B test on offer angle and CTA.
2. Improve delivery process where time is wasted.
3. Build one automation that saves manual hours weekly.
4. Close first client and run onboarding with template.
5. Publish first documented win.
