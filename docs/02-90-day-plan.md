# 90-Day Agency Execution Plan

## Phase 1: Foundation (Days 1-14)

1. Lock one ICP and one flagship offer.
2. Define package scope, pricing floor, and delivery boundaries.
3. Build standard sourcing + enrichment + QA workflow.
4. Create 2 campaign briefs and 2 sequence variants.
5. Finalize onboarding and weekly reporting templates.

## Phase 2: Client and Pipeline Build (Days 15-45)

1. Run outbound for LeadGent itself to close first clients.
2. Launch pilot delivery for first 1-2 clients.
3. Track delivery and outbound KPIs weekly.
4. Capture repeatable process steps into templates.
5. Publish one mini case study with quantifiable results.

## Phase 3: Optimization and Scale (Days 46-90)

1. Remove low-performing segments and offers.
2. Expand only top-performing service lane.
3. Improve margin via SOPs and automation.
4. Introduce one expansion service to existing clients.
5. Implement monthly account reviews and retention plan.

## Weekly Deliverables (Non-Negotiable)

- One client outcome improved
- One internal process documented or automated
- One business development action completed
